# 美股基金涨跌助手

> 实时追踪美股三大指数及对应 QDII 基金涨跌情况，帮助投资者预判次日 A 股基金走势。

## 功能

- 美股三大指数（纳斯达克、标普500、道琼斯）+ 人民币汇率实时涨跌
- 对应 QDII 基金"参考涨跌"列表，揭示指数涨 ≠ 基金涨的真实情况
- 深色主题 UI，适配移动端
- 点击指数卡片可筛选对应基金
- 支持涨跌排序切换
- 自动每 5 分钟刷新数据

## 技术栈

- 纯静态页面：HTML + CSS + JavaScript
- 数据：JSON 文件 + Yahoo Finance / 汇率 API
- 部署：GitHub Pages

## 项目结构

```
├── index.html      # 主页面
├── style.css       # 样式（深色主题）
├── script.js       # 数据获取与渲染逻辑
├── data.json       # 基金与指数数据（缓存）
├── update_data.py  # 数据更新脚本（定时执行）
└── README.md
```

## 数据来源

- 美股指数：Yahoo Finance API
- 汇率：ExchangeRate-API
- QDII 基金：天天基金（手动维护 + 自动估值）
